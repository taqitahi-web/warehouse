🔥 Firebase Setup

EPFBD Warehouse Management System — Cloud Sync Configuration

📋 Firebase Project তৈরির ধাপ

1

Firebase Console যান: console.firebase.google.com — Google account দিয়ে login করুন

2

New Project: "Add project" → Project name: epfbd-wms → Analytics off করুন → "Create project"

3

Realtime Database চালু করুন: Left sidebar → Build → Realtime Database → "Create Database" → Location: us-central1 → Start in test mode → Enable

4

Web App যোগ করুন: Project Overview → "</>" icon → App nickname: EPFBD WMS → Register app → নিচে config copy করুন

5

Database Rules সেট করুন: Realtime Database → Rules → এই rule paste করুন:

{ "rules": { "wms": { ".read": true, ".write": true } } }

⚠️ Test mode — পরে PIN-based auth যোগ করা যাবে

⚙️ Firebase Config এখানে পেস্ট করুন

⚠️ Firebase Console → Project Settings → Your apps → SDK setup → Config object থেকে values copy করুন
Firebase config এখানে দিন এবং Test Connection চাপুন...
